# Community Menu

GNOME Shell 50 applications menu used by the Community Layout Switcher.

The extension is a GPL-2.0-or-later derivative of Zorin Menu 5.3.5. It
keeps the application-list, application-grid, and Mint-style layouts while
using its own UUID, settings schema, name, and neutral artwork. Only the stock GNOME
panel and Dash to Panel are supported.

Layout mapping:

- `APPS_ONLY`: Classic
- `APP_GRID`: Desk-UX
- `MINT`: Hybrid

See `UPSTREAM.md` and `COPYING` for provenance and licensing.
